# 🤖 AI Chatbot using Lovable + n8n + Google Gemini API

This project is an AI-powered chatbot that combines a clean frontend interface with a powerful no-code/low-code automation backend. It uses Google's Gemini model for generating intelligent, real-time responses.

The goal of this project was to build a practical, scalable chatbot system using modern tools without relying on heavy backend frameworks.

---

## 🚀 Features

- 💬 Real-time AI chatbot interface
- 🔗 Seamless integration with Google Gemini API
- ⚙️ Automated workflows using n8n
- 🎯 Customizable prompts and responses
- 🧩 Modular and easy-to-extend architecture
- 🌐 Lightweight and user-friendly UI (Lovable)

---

## 🛠️ Tech Stack

### Frontend
- Lovable (for UI design and interaction)

### Backend / Automation
- n8n (workflow automation)

### AI Model
- Google Gemini API (for natural language responses)

---

## 🧠 How It Works

1. User enters a message in the chatbot UI.
2. The request is sent to an n8n webhook.
3. n8n processes the input and forwards it to the Gemini API.
4. Gemini generates a response.
5. n8n sends the response back to the UI.
6. The chatbot displays the output instantly.
